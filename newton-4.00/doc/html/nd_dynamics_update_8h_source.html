38 // the solver is a RK order 4, but instead of weighting the intermediate derivative by the usual 1/6, 1/3, 1/3, 1/6 coefficients
39 // I am using 1/4, 1/4, 1/4, 1/4.
40 // This is correct.The weighting coefficients of any RK method comes from an arbitrary criteria
41 // solving for a set of linear equation on the coefficients.
42 // The standard coefficients just happen to lead to an accurate result because they are optimal
43 // with respect to a second order solution.For differential equations of higher order, then is not
44 // clear if the traditional weigh factors are any better than any other set of weighting factors.
45 // A different set of coefficient generates results that are not much different than the optimal set,
46 // but it allows for simpler calculation of the intermediate derivatives and also for less intermediate memory.
47 // For more detail on the derivation of the Runge Kutta coefficients you can go to:
48 // http://pathfinder.scar.utoronto.ca/~dyer/csca57/book_P/node51.html
